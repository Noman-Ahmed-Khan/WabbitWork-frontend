# Team Task Manager - Frontend

A modern, responsive team task management application built with React, Vite, Tailwind CSS, and DaisyUI.

## Features

- **Authentication**: Secure session-based authentication
- **Dashboard**: Overview of tasks with statistics
- **Team Management**: Create, edit, and manage teams
- **Member Management**: Add/remove members with role-based access
- **Task Management**: Create, assign, filter, and track tasks
- **Responsive Design**: Works seamlessly on all devices
- **Dark Mode**: Built-in theme switcher
- **Liquid Morphic UI**: Modern, clean design system

## Tech Stack

- **React 18** - UI framework
- **Vite** - Build tool
- **Tailwind CSS 4** - Styling
- **DaisyUI** - Component library
- **Axios** - HTTP client
- **React Router** - Navigation
- **Context API** - State management
